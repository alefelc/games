# ¿Te animás? — frontend dinámico v2.7.0

Al abrir o actualizar la app se leen los registros publicados del juego.

Aparecen automáticamente:

- sexos;
- niveles;
- mazos activos;
- modos;
- elementos;
- juguetes;
- etiquetas;
- cartas;
- relaciones de cartas con mazos, etiquetas, elementos y juguetes.

Una carta nueva debe estar publicada, vinculada al juego, vinculada a un nivel
y relacionada con un mazo activo para participar cuando los mazos están seleccionados.

Si no hay conexión, se usa la última copia guardada.
